## Overview
### About

Comfy is a MERN fullstack furniture Ecommerce project that has the following features:

- Regiseteration / logging-in system
- Shopping cart
- Product details
- Checkout process
- User account
- Admin dashboard
- Search for products
- Filter products by price, categoty and brand
- Contact form
